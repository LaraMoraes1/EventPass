# Checklist da apresentação

1. Subir PostgreSQL online e executar `database/schema.sql`.
2. Subir a API com `mvn spring-boot:run`.
3. Confirmar que `GET /api/events` retorna JSON.
4. Ajustar `BASE_URL` no Android para o IP correto.
5. Instalar o app em dois celulares.
6. Celular 1: entrar como participante e abrir QR Code.
7. Celular 2: entrar como administrador e abrir Scanner.
8. Escanear uma vez para entrada e outra vez para saída.
9. Mostrar Dashboard administrativo e Histórico.

Plano B para demonstração: se a rede da faculdade oscilar, use hotspot de um celular e conecte notebook + celulares na mesma rede.
